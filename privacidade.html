Política de Privacidade

"Sua segurança é o nosso código principal."

1. Compromisso Zero-Log

O Descomplic.AI não armazena, salva ou compartilha os documentos enviados para análise. Assim que o processamento é concluído e a resposta enviada ao seu navegador, o arquivo é permanentemente deletado de nossa memória temporária.

2. Processamento de Dados

Os documentos são enviados via protocolo HTTPS (criptografado) para a API do Google Gemini para a realização da análise via IA. Nenhum dado pessoal identificável é coletado durante este processo.

3. Cookies e Rastreamento

Não utilizamos cookies de publicidade. Utilizamos apenas métricas anônimas para entender o volume de acessos e garantir a estabilidade do servidor.
